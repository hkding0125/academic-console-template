cv import helper

Upload a PDF or DOCX CV, review the extracted content, and generate HTML snippets for this template.

This tool is best for creating a first draft. Always review the extracted content before publishing.

upload

Supported formats: .pdf and .docx. PDF extraction depends on how selectable the text is in your file. Scanned-image PDFs may require manual copy/paste cleanup.

Waiting for file upload.

Tips for better results
  • Use a text-based PDF, not a scanned image PDF.
  • Keep section headings explicit: Education, Experience, Publications, Awards, Contact.
  • Publication lists with DOI links are easier to recognize.
  • You can edit every extracted field before generating HTML.
