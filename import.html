cv import wizard

Upload a PDF or DOCX CV, review a few key fields, and download ready-to-use pages for this template.

This wizard is designed for a first draft. Always review the generated result before publishing.

Step 1

Upload your CV

Start with a .pdf or .docx file. Text-based PDFs work much better than scanned-image PDFs.

Waiting for file upload.

Tips for better results
  • Use a text-based PDF, not a scanned image PDF.
  • Keep section headings explicit: Education, Experience, Publications, Awards, Contact.
  • Publication lists with DOI links are easier to recognize.
  • You can still refine the extracted result before exporting pages.
