cv 导入助手

上传 PDF 或 DOCX 格式的 CV，检查提取结果，并为这个模板生成可复制的 HTML 片段。

这个工具更适合生成第一版草稿。正式发布前请务必手动检查内容。

上传

支持格式：.pdf 和 .docx。如果 PDF 文本本身不可选中，提取效果会明显下降，扫描版 PDF 通常需要手动整理。

等待上传文件。

提高识别效果的小建议
  • 优先使用文本可选中的 PDF，而不是扫描图片版 PDF。
  • CV 里最好有清晰的小节标题，比如 Education、Experience、Publications、Awards、Contact。
  • 带 DOI 的论文条目通常更容易识别。
  • 所有提取字段都可以在生成 HTML 前手动修改。
